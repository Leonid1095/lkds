# Дорожная карта развития

## Этап 1 — MVP (текущий)
- Почасовое бронирование переговорки.
- Визуальное расписание занятости.
- Контактные поля ФИО и Telegram.
- Раздел полезных ссылок.
- Веб-форма заявок по 1С CRM.

## Этап 2 — Удобство и контроль
- Фильтры по дате/неделе.
- Отмена бронирования по коду/ссылке подтверждения.
- Валидация бизнес-правил (минимальная/максимальная длительность).
- Экспорт расписания в CSV/Excel.

## Этап 3 — Интеграции
- Интеграция с Telegram-ботом (уведомления о бронях).
- Вебхуки в корпоративные системы.
- Единая витрина заявок (переговорки + 1С CRM).

## Этап 4 — Масштабирование
- Переход с JSON-хранилища на PostgreSQL.
- Роли (сотрудник/администратор) при необходимости.
- История изменений и аудит действий.
- Отдельные правила бронирования для разных комнат.

## Этап 5 — Аналитика
- Нагрузка переговорок по дням/часам.
- Популярные слоты и коэффициент утилизации.
- SLA по заявкам 1С CRM.
