# Сайт бронирования переговорок (MVP)

Простой внутренний сайт без регистрации:
- просмотр почасовой занятости переговорок,
- создание брони по ФИО + Telegram,
- блок полезных корпоративных ссылок,
- форма заявки по проблемам 1С CRM.

## Быстрый старт

1. Установите зависимости:
   ```bash
   npm install
   ```
2. Создайте `.env` на основе `.env.example`:
   ```bash
   copy .env.example .env
   ```
3. Запустите проект:
   ```bash
   npm run dev
   ```
4. Откройте `http://localhost:3000`.

## Где менять временный домен

- В файле `.env` параметр:
  - `PUBLIC_BASE_URL=https://your-domain.example.com`

Это значение отображается в интерфейсе и может быть быстро заменено при переносе на постоянный домен.

## Где менять переговорки и ссылки

- Переговорки: `data/rooms.json`
- Полезные ссылки: `data/links.json`

Чтобы добавить новую переговорку, просто добавьте новый объект в `data/rooms.json`.

## Ограничения MVP

- Хранилище на JSON-файлах (без БД).
- Нет авторизации и прав доступа.
- Удаление/редактирование брони пока не включено.

## Что уже заложено для расширения

- Мультикомнатность (через `roomId`).
- Отдельный модуль заявок 1С CRM.
- Конфиг домена через `.env`.
